Criação de um Data Warehouse usando Redshift e visualizando os dados com Looker Studio

Criação e carga de dados para um Data Warehouse utilizando as ferramentas AWS Redshift e AWS S3 e depois realizando visualizações com o Looker Studio. Projeto desenvolvido no curso de Engenharia de Dados em Big Data.

O objetivo deste projeto foi criar um Data Warehouse utilizando a ferramenta AWS Redshift e logo após isso conectar o DW na ferramente Looker Studio e criar um pequeno dashboard para visualizar os dados. Aqui será registrado o passo a passo seguido e comentado de maneira detalhada.

O primeiro ponto foi criar o DW no Redshift. Para isso foi preciso de uma conta na plataforma da AWS e logo em seguida foi selecionado a ferramenta do Redshift e criado um database chamado "ed" como mostra a imagem a seguir.

Logo após isso, foi usado um script disponibilizado pelo professor do curso para criação de 5 tabelas (Produtos, Vendedores, ItensVenda, Vendas e Clientes) dentro do DW, isso serve para que possamos importar os dados que estão armazenados em um bucket no AWS S3 dentro do nosso DW. A imagem a seguir mostra um pouco do script.

O próximo passo foi fazer a importação dos dados salvos em um bucket no AWS S3 para dentro das tabelas criadas no Redshift. As duas imagens a seguir mostram os arquivos .csv salvos no bucket e uma parte do código para copiar os dados desses arquivos nas tabelas, respectivamente.

Esse trecho do código teve que ser utilizado 5 vezes, uma para cada tabela e arquivo no S3, única coisa que diferenciava um do outro era o caminho do arquivo .csv dentro do bucket (as credenciais não estão censuradas pois o bucket já foi excluído).

Depois dessas etapas, foi a vez de realizar os joins entre as tabelas e criar o nosso modelo análitico (que será utilizado no Looker Studio). Foi criado então a tabela "fatovendas" como pode ser observado na figura abaixo.
